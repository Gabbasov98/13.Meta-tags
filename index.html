С помощью даных попей можно автоматически сгенерировать сео данные для товаров этой категории и ее под категорий Вставки типа {Sname} заменятся на соответствующие значения этого товара Возможные варианты вставок перечислены ниже:

• {Sname} - Название товара

• (Sarticle) - Артикул товара

• {$short_description} - Короткое описание

• {Scategory} - Название категории

• {Sbrand} - Название бренда

• {Sprice} - Цена товара

• {$old_price} - Старая цена товара
